Platform User Types

System Administrator

Manages platform configuration, WhatsApp setup, and system monitoring

Primary Tasks:

WhatsApp Web connection, user management, system configuration, security oversight

Message Operator

Handles daily message monitoring, ticket management, and emergency response

Primary Tasks:

Real-time monitoring, flagged message review, ticket creation, bulk messaging

Data Analyst

Analyzes sentiment trends, generates reports, and monitors analytics

Primary Tasks:

Data analysis, report generation, trend identification, export to external systems

Field Coordinator

Manages groups, coordinates operations, handles public communication

Primary Tasks:

Group management, member coordination, bulk operations, field communication

Administrator Workflow

Initial Platform Setup

1
Platform Access

Access the management platform and review the main dashboard interface.

2
WhatsApp Setup

Navigate to WhatsApp Setup page and choose connection method (Web or API).

3
QR Connection

Generate QR code, scan with phone, establish WhatsApp Web connection.

4
User Management

Create user accounts, assign roles, configure permissions and access levels.

5
ML Configuration

Configure ML models, set flagging thresholds, customize alert priorities.

6
Monitor Operations

Monitor platform performance, review logs, ensure optimal operation.

Daily Operator Workflow

Daily Operations Cycle

1
Dashboard Review

Check overnight alerts, review system status, assess pending items.

2
Flagged Messages

Review AI-flagged messages, prioritize by urgency, assess content validity.

3
Ticket Creation

Create tickets from flagged messages, assign to teams, track progress.

4
Real-time Monitor

Monitor live message feed, respond to alerts, handle emergencies.

5
Bulk Operations

Send scheduled messages, broadcast announcements, manage groups.

6
Daily Summary

Complete shift summary, update status, prepare handover notes.
